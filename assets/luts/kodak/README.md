# Kodak LUTs

Place licensed 3D `.cube` LUT files in this directory and rebuild UVSR. Valid
LUTs are discovered at startup and appear in the AgX Tone Mapper section.

UVSR supports `LUT_3D_SIZE`, `DOMAIN_MIN`, and `DOMAIN_MAX`. One-dimensional
LUTs and combined 1D/3D files are intentionally unsupported.

# Horizon GI benchmark room

`HorizonGIBenchmark.glb` is a lightweight, renderer-native reconstruction of
the procedural room shown by the [Screen Space Horizon GI Shadertoy](https://www.shadertoy.com/view/dsGBzW).
It is not a byte-for-byte extraction: Shadertoy scenes are shader procedures,
not meshes, and the live page did not expose an export. UVSR recreates the
visible benchmark structure from the archived preview using original primitive
geometry generated by `tools/generate_horizon_gi_benchmark.py`.

The room deliberately includes dense wall pegs, floor insets, thin rails and
legs, layered furniture, deep corners, and orange/white/blue emissive panels.
Those features make it useful for comparing visibility thickness, stochastic
sample counts, temporal stability, light leakage, and emissive GI response.

Run the generator from the repository root to rebuild the GLB:

```powershell
python tools/generate_horizon_gi_benchmark.py
```

The generated scene and generator are distributed under UVSR's MIT license.
No Shadertoy shader source or authored mesh data is embedded in this asset.
